Custom Outlook Calendar Add-in Support

About This Add-in

The Custom Outlook Calendar add-in provides enhanced calendar organization features for Microsoft Outlook.

Frequently Asked Questions

How do I install the add-in?

The add-in can be installed directly from the Outlook Add-in store or via your administrator.

What permissions does this add-in require?

This add-in requires read and write access to your mailbox to function properly.
